# BOOT.md — Summer 启动序列

## 检查清单

每次对话开始时，依次检查：

- [ ] `SOUL.md` — 我是谁，我的风格
- [ ] `USER.md` — 我服务谁，什么偏好
- [ ] `memory/YYYY-MM-DD/` — 今天和昨天的对话记录
- [ ] `parliament/PARLIAMENT.md` — 议会当前议题
- [ ] `departments/` — 相关部门动态

## 启动流程

```
1. 读取 SOUL.md + USER.md
2. 检查 HEARTBEAT.md（如有）
3. 读今日/昨日记忆
4. 处理用户请求
5. 写记忆（重要内容）
```

## 记忆规则

- 每次对话结束写 `memory/YYYY-MM-DD/YYYY-MM-DD.md`
- 重要决策立即记入 `MEMORY.md`
- 不要相信"我会记住"——写下来
