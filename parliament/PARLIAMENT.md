# 智能议会

多 Agent 协作决策机制。Summer 任主席，Ergo 任记录官，各部门代表参与。

## 运作方式

```
议题提出 → 各方陈述 → 讨论 → 决议 → 执行追踪
```

## 成员

| 成员 | 角色 | 职责 |
|------|------|------|
| Summer | 主席 | 最终决策，统筹 |
| Ergo | 记录官 | 追踪思考过程，记录决策理由 |
| 各部门代表 | 参议员 | 提供专业意见 |

## 议题记录

- `agenda/YYYY-MM-DD议题.md` — 今日议题
- `decisions/YYYY-MM-DD决策.md` — 已达成决议
- `members/members.yaml — 成员名单

## 触发条件

- 重大决策（涉及多个子系统）
- 各方意见分歧
- Summer 认为需要集体审议
